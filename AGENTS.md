# AGENTS.md - Hanzi Survivors（汉字幸存者）

## 项目定位

竖屏 PixiJS 吸血鬼幸存者-like 游戏。所有单位使用汉字表示——玩家是「我」，敌人是单字/双字精英/四字BOSS，技能由笔画（横竖撇捺等）合成文字释放。

## 技术栈

- Vite + TypeScript + PixiJS ^8.x
- 竖屏 540×960 逻辑分辨率
- 虚拟摇杆 + 触屏/鼠标操作
- GitHub Pages 部署

## 协作规则

1. **文档优先**：玩法/数值变更先改 `docs/game/`，再改 `src/`
2. **分步推进**：功能不要一次做太多，拆成小步骤逐步确认
3. **先计划再动手**：有歧义或复杂度较高时，先出计划跟阿燃确认
4. **提交规范**：功能完成后审查并 Git 提交，push 到 GitHub master

## 核心玩法设计

### 笔画收集与技能合成
- 敌人死亡掉落笔画（横、竖、撇、捺、点等）
- 玩家拾取笔画，系统自动匹配合成技能文字
- 技能文字显示在下方技能栏，玩家点击释放
- 相同文字自动合并升级：火×2→炎，炎+火→焱，焱+火→燚
- 释放后技能被消耗，需要重新收集

### 技能树
- `十` = 一 + 丨（穿透十字镖）
- `卍` = 一×3 + 丨×3（回旋镖）
- `小` = 点 + 撇 + 撇 + 捺（小火花范围伤害）
- `火` = 小 + 人（火球）
- `炎` = 火 + 火（双重火球）
- `焱` = 炎 + 火（三重火球）
- `燚` = 焱 + 火（四火大爆炸）

### 波次
- 共 20 波，敌人数量递增
- 第 5 波起出现精英（双字敌人）
- 第 10、20 波出现 BOSS（四字敌人）
- 全灭敌人胜利，HP 归零失败
